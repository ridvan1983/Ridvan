# Ridvan: AI-Powered Full-Stack Development Platform

Ridvan is an AI-powered full-stack development platform that allows you to prompt, run, edit, and deploy production-grade applications directly from your browser—no local setup required. Built on top of the proven bolt.new open-source foundation, Ridvan provides enhanced AI capabilities with a custom development experience.

## What Makes Ridvan Different

Traditional AI assistants like Claude, v0, etc are incredible- but you can't install packages, run backends or edit code. That's where Ridvan stands out:

- **Full-Stack in the Browser**: Ridvan integrates cutting-edge AI models with an in-browser development environment powered by **StackBlitz's WebContainers**. This allows you to:
  - Install and run npm tools and libraries (like Vite, Next.js, and more)
  - Run Node.js servers
  - Interact with third-party APIs
  - Deploy to production from chat
  - Share your work via a URL

- **AI with Complete Environment Control**: Unlike traditional dev environments where the AI can only assist in code generation, Ridvan gives AI models **complete control** over the entire environment including the filesystem, node server, package manager, terminal, and browser console. This empowers AI agents to handle the entire app lifecycle—from creation to deployment.

## Foundation & License

Ridvan is built upon the excellent bolt.new open-source codebase from StackBlitz and maintains the same **MIT License**. All core WebContainer and AI integration technologies remain unchanged, ensuring compatibility and reliability while adding Ridvan-specific enhancements.

## Tips and Tricks

Here are some tips to get the most out of Ridvan:

- **Be specific about your stack**: If you want to use specific frameworks or libraries (like Astro, Tailwind, ShadCN, or any other popular JavaScript framework), mention them in your initial prompt to ensure Ridvan scaffolds the project accordingly.

- **Use the enhance prompt icon**: Before sending your prompt, try clicking the 'enhance' icon to have the AI model help you refine your prompt, then edit the results before submitting.

- **Scaffold the basics first, then add features**: Make sure the basic structure of your application is in place before diving into more advanced functionality. This helps Ridvan understand the foundation of your project and ensure everything is wired up right before building out more advanced functionality.

- **Batch simple instructions**: Save time by combining simple instructions into one message. For example, you can ask Ridvan to change the color scheme, add mobile responsiveness, and restart the dev server, all in one go saving you time and reducing API credit consumption significantly.

## FAQs

**Where do I sign up for a paid plan?**  
Ridvan is free to get started. If you need more AI tokens or want private projects, you can purchase a paid subscription in your Ridvan settings, in the lower-left hand corner of the application. 

**What happens if I hit the free usage limit?**  
Once your free daily token limit is reached, AI interactions are paused until the next day or until you upgrade your plan.

**Is Ridvan in beta?**  
Yes, Ridvan is in beta, and we are actively improving it based on feedback.

**How can I report Ridvan issues?**  
Check out the [Issues section](https://github.com/stackblitz/bolt.new/issues) to report an issue or request a new feature. Please use the search feature to check if someone else has already submitted the same issue/request.

**What frameworks/libraries currently work on Ridvan?**  
Ridvan supports most popular JavaScript frameworks and libraries. If it runs on StackBlitz, it will run on Ridvan as well.

**How can I add make sure my framework/project works well in Ridvan?**  
We are excited to work with the JavaScript ecosystem to improve functionality in Ridvan. Reach out to us via [hello@stackblitz.com](mailto:hello@stackblitz.com) to discuss how we can partner!
